## 📌 Sobre o Projeto
Este projeto foi desenvolvido para aplicar conhecimentos de **Data Science** e **Machine Learning** em um cenário real que me desperta muito interesse: o setor de fitness e academias. 

O objetivo principal é prever o **Churn** (evasão de alunos), identificando quais membros têm maior probabilidade de cancelar sua matrícula. Com essas informações, uma academia pode agir preventivamente com campanhas de retenção.

## 🗂️ Base de Dados
Os dados utilizados foram obtidos no **Kaggle**:
* **Dataset:** [Churn Prediction Gym Members](https://www.kaggle.com/datasets/hassaanabid/churn-prediction-gym-members-dataset)
* **Fonte:** Hassaan Abid

O conjunto de dados inclui informações como idade, gênero, tipo de plano, frequência de treinos, calorias queimadas e a data da última visita.

## 🚀 Tecnologias e Conceitos Aplicados
Neste projeto, percorri as etapas essenciais de um fluxo de Data Science:

1.  **Limpeza de Dados:** Tratamento de valores ausentes (NaN) e conversão de tipos de dados.
2.  **Engenharia de Atributos:** Transformação de datas em formatos numéricos (Timestamps) e criação de novas métricas.
3.  **Análise Exploratória (EDA):** Visualização de padrões através de correlações, boxplots e gráficos de dispersão usando `Seaborn` e `Matplotlib`.
4.  **Machine Learning:** Aplicação de algoritmos de **Classificação** (em desenvolvimento).
5.  **Métricas de Avaliação:** Foco em métricas como Recall e Matriz de Confusão para entender o erro do modelo.
